# raycast-commands-kit 介绍
🚀 提升工作效率的工具箱：Raycast脚本集合，自动化日常高频操作，消灭窗口/应用/链接的高并发繁琐切换，将系列步骤缩短为一次按键，可将常见操作压缩至 1 秒内，可减少约90% 的上下文切换，单次任务平均节省15秒。保护心流不间断，提高工作效率。

# 快速开始
```bash
# 克隆
git clone https://github.com/ismosi/raycast-commands-kit.git
```
🎯 将 scripts/ 下的 .sh 文件添加到 Raycast → Settings → Extensions → Script Commands 的脚本目录中。

# 核心命令
## Script Commands（脚本命令）
 | 命令 | 触发词 | 效果 |
 | :--- | :--- | :--- |
| 🎯 一键专注模式 | `fm`(focus mode 命令) | 一键自动开启瞬时静音、关闭通讯软件、切换深色模式、启动番茄钟，最后自动将Obisidian等笔记软件与、Trae等代码IDE（未来可自由扩充）在固定桌面打开，保留原多桌面 APP 隔离 |
| 🔧 Git状态速查 | `gs`(git status 命令) | 任意目录查看git状态，自动定位项目根目录 |
| 🚀 启动Dev服务器 |` ns`(npm start 命令)	|  自动检测 `package.json`，自动打开终端启动服务器 |

描述：任何可选中文字的地方（只要这个页面有可输入的地方，系统级调用raycast弹窗即可输入（无需切换任何界面），因为可以自己输入再选中），尤其是终端中运行各种命令的英文界面、各种陌生 APP 的配置界面，或者不可复制的英文界面，都可以做到在该页面内（无需切换）进行翻译，而且翻译是从词源级别开始拆解，任何市面上都很难看到这种从词源级别演化到现在的各种衍生常见流行含义的变迁领悟形式的学习，配合中英文双语的各种动宾例句做到了极致的用户体验，真真正正的解决了在工作中的千百个页面遇到生词难词总是翻来覆去查词又容易忘记，又在高并发的工作中不容易记录生词的痛点，有了这个工具，在实践中学习复习成为可能，不需要死记硬背，在实践中，自然就可以习得，因为这种实践性学习的成本被该工具拉到极低

| 🇨🇳 翻译成中文 | 英→中，一键词源级拆解 | 读英文文档，理解词汇演化 |
| 🇬🇧 翻译成英文 | 中→英，一键词源级拆解 | 写注释、写文档、国际化文案——精准选词，从演化脉络理解用法 |
| 🏢 翻译成职场黑话 | 一键日常→专业协作语言 | 周报、方案、跨部门沟通——提升专业密度，建立共识效率 |

| 📝 翻译成Git Commit | 代码变更→规范提交信息 | 省去自然语言到Conventional Commit的翻译成本，中英文可选，简洁/详细可调 |

# 设计原则  
> 用理性压缩无意义的消耗，为感性保留流动的空间。

> 工具适应人，是为了让人不必在机械重复中耗尽自己——省下来的注意力，可以去应对真正需要人的事情：犹豫、权衡、博弈、判断、创造、连接、感受、影响、超越、迭代......
- 保护心流：3次按键内完成，心流中断成本趋近于零
- 零上下文切换：结果即时呈现，不跳出当前工作场景
- 自适应兜底：传参精准命中 → 无参递归探测 → 失败清晰提示
